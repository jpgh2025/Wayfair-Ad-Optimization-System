How to Use WSP Optimizer
Required Files:
  • Campaign Performance Report - Campaign metrics and ROAS
  • Keyword Optimization Report - Keyword performance data
  • Search Term Research Report - Search query analysis
  • Product Performance Report - Product-level metrics
Process:
  1. Upload all 4 CSV files from Wayfair
  2. Click "Optimize" to start analysis
  3. Wait for processing (usually 1-2 minutes)
  4. Download optimized bulk upload files
  5. View interactive dashboard for insights
